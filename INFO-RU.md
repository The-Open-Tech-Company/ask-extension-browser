# Браузерный помощник - Расширение для Chrome/Chromium

Расширение для браузеров на базе Chromium, которое помогает ускорить работу в браузере с функциями перевода текста, поиска на странице и объяснения с помощью ИИ.

## Основные функции

### 1. Поиск текста на странице
- Откройте popup расширения
- Введите текст для поиска в поле поиска
- Настройте параметры:
  - **Точное совпадение** - находит только 100% совпадения
  - **Учитывать морфологию** - учитывает склонения и спряжения слов (для русского языка)
- Нажмите "Поиск" для поиска совпадений
- Используйте кнопки "Назад" и "Вперед" для навигации по результатам
- Нажмите "Очистить выделения" для удаления всех выделений

### 2. Перевод текста
- Выделите нужный текст на странице
- Нажмите правой кнопкой мыши → Расширение → **Перевести**
- Рядом с текстом появится всплывающее окно с переводом

**Настройки перевода:**
- По умолчанию все иностранные языки переводятся на русский
- С русского языка перевод идет на английский (настраивается)
- Настройки можно изменить в настройках расширения

### 3. Перевод всей страницы
- Нажмите на иконку расширения
- Нажмите "Перевести страницу" в popup
- Вся страница будет переведена с помощью Google Translate

### 4. Объяснение текста с помощью ИИ
- Выделите текст на странице
- Нажмите правой кнопкой мыши → Расширение → **Объяснить**
- Откроется новое окно с чатом ИИ, который объяснит текст и ответит на вопросы
- Или откройте чат через popup → "Открыть чат с ИИ"

**Настройки ИИ:**
- Выбор модели: **DeepSeek** (с поддержкой поиска в интернете и мышления) или **ChatGPT** от OpenAI
- Настройка API ключа
- Настройка температуры ответа (0.0 - 2.0)
- Системный промпт (опционально)
- Режим разработчика для отображения информации об использовании токенов

**Особенности DeepSeek:**
- В чате доступны кнопки для включения функций:
  - **Поиск в интернете** - позволяет ИИ искать информацию в интернете
  - **Мышление** - включает расширенное мышление для сложных задач

**Модели ChatGPT:**
- GPT-4
- GPT-4 Turbo
- GPT-3.5 Turbo
- GPT-4o
- GPT-4o Mini

### 5. Шаблоны промптов
- Предустановленные шаблоны для быстрого взаимодействия с ИИ
- Поддержка пользовательских шаблонов
- Быстрый доступ через короткие названия
- Шаблоны можно редактировать и удалять (только пользовательские)

## Установка

1. Скачайте или клонируйте репозиторий
2. Создайте иконки расширения (см. раздел "Создание иконок")
3. Откройте браузер Chrome/Chromium
4. Перейдите в `chrome://extensions/` (или `edge://extensions/` для Edge)
5. Включите "Режим разработчика" (Developer mode)
6. Нажмите "Загрузить распакованное расширение" (Load unpacked)
7. Выберите папку с расширением

## Создание иконок

Расширению необходимы иконки размером 16x16, 48x48 и 128x128 пикселей в формате PNG.

Вы можете:
1. Использовать онлайн-генераторы иконок
2. Создать иконки в графическом редакторе
3. Использовать готовые иконки из интернета

Поместите файлы:
- `icons/icon16.png` (16x16 пикселей)
- `icons/icon48.png` (48x48 пикселей)
- `icons/icon128.png` (128x128 пикселей)

## Настройка

### Настройки перевода
1. Нажмите на иконку расширения в панели инструментов
2. Перейдите на вкладку "Перевод"
3. Выберите язык перевода по умолчанию для иностранных языков
4. Выберите язык перевода с русского
5. Нажмите "Сохранить"

### Настройки ИИ
1. Нажмите на иконку расширения в панели инструментов
2. Перейдите на вкладку "Настройки AI"
3. Выберите модель ИИ (DeepSeek или ChatGPT)
4. Если выбрана модель ChatGPT, выберите конкретную модель из списка
5. Введите API ключ:
   - Для DeepSeek: получите ключ на https://platform.deepseek.com/api_keys
   - Для ChatGPT: получите ключ на https://platform.openai.com/api-keys
6. Настройте температуру ответа (0.0 - 2.0)
7. При необходимости введите системный промпт
8. Включите режим разработчика при необходимости (показывает использование токенов)
9. Нажмите "Сохранить"

### Шаблоны промптов
1. Нажмите на иконку расширения в панели инструментов
2. Перейдите на вкладку "Шаблоны"
3. Нажмите "Добавить шаблон" для создания нового шаблона
4. Заполните:
   - Название (обязательно)
   - Промпт (обязательно)
   - Описание (опционально)
   - Краткое название (опционально, максимум 15 символов)
5. Нажмите "Сохранить"
6. Редактируйте или удаляйте пользовательские шаблоны по необходимости

## Использование

### Поиск на странице
1. Нажмите на иконку расширения в панели инструментов
2. В popup введите текст для поиска
3. Настройте параметры поиска (точное совпадение, морфология)
4. Нажмите "Поиск" для поиска совпадений
5. Используйте кнопки навигации для перехода между результатами
6. Нажмите "Очистить выделения" для удаления всех выделений

### Перевод текста
1. Выделите текст на любой веб-странице
2. Нажмите правой кнопкой мыши
3. В контекстном меню выберите "Расширение" → "Перевести"
4. Рядом с текстом появится окно с переводом

### Перевод всей страницы
1. Нажмите на иконку расширения
2. Нажмите "Перевести страницу" в popup
3. Дождитесь перевода страницы

### Объяснение текста
1. Выделите текст на любой веб-странице
2. Нажмите правой кнопкой мыши
3. В контекстном меню выберите "Расширение" → "Объяснить"
4. Откроется окно чата с ИИ
5. ИИ автоматически объяснит выделенный текст
6. Вы можете задавать дополнительные вопросы в чате
7. Для DeepSeek можно включить функции "Поиск в интернете" и "Мышление"

### Управление расширением
1. Нажмите на иконку расширения в панели инструментов
2. Используйте переключатель для включения/выключения расширения
3. Нажмите "Открыть чат с ИИ" для открытия чата без выделенного текста
4. Нажмите "Настройки" для настройки перевода и ИИ

## Требования

- Браузер на базе Chromium (Chrome, Edge, Brave, Opera и т.д.)
- API ключ для выбранной модели ИИ (DeepSeek или OpenAI)
- Интернет-соединение для работы функций перевода и ИИ

## Технические детали

- **Версия манифеста**: 3
- **Версия расширения**: 2.3.5
- **Перевод**: Использует публичный API Google Translate
- **DeepSeek API**: https://api.deepseek.com/v1/chat/completions
- **OpenAI API**: https://api.openai.com/v1/chat/completions
- **Хранилище**: Использует `chrome.storage.sync` для настроек и `chrome.storage.local` для истории чатов и шаблонов

## Устранение неполадок

### "Функции недоступны. Проверьте API ключ"
- Убедитесь, что вы ввели правильный API ключ в настройках
- Проверьте, что ключ активен и имеет необходимые разрешения
- Для DeepSeek убедитесь, что используете правильный формат ключа

### Перевод не работает
- Проверьте интернет-соединение
- Убедитесь, что сайт не блокирует запросы к Google Translate
- Попробуйте выделить текст заново

### Окно чата не открывается
- Убедитесь, что браузер разрешает всплывающие окна для этого сайта
- Проверьте настройки блокировки всплывающих окон в браузере

### Поиск не работает
- Убедитесь, что расширение включено (проверьте переключатель в popup)
- Попробуйте обновить страницу
- Проверьте, что content scripts загружены (откройте консоль браузера)

### Ответы ИИ медленные
- Проверьте интернет-соединение
- Убедитесь, что API ключ правильный
- Для ChatGPT проверьте, что используете правильную модель и у вас достаточно кредитов
