# Браузерный помощник - Расширение для Chrome/Chromium

Расширение для браузеров на базе Chromium, которое помогает ускорить работу в браузере с функциями перевода текста, поиска на странице и объяснения с помощью ИИ.

## Основные функции

- **Поиск текста на странице** с поддержкой морфологии для русского языка
- **Перевод текста** через контекстное меню
- **Объяснение с помощью ИИ** через DeepSeek или ChatGPT
- **Перевод всей страницы**
- **Шаблоны промптов** для быстрого взаимодействия с ИИ

## Установка

1. Скачайте или клонируйте репозиторий
2. Создайте иконки расширения (16x16, 48x48, 128x128 PNG)
3. Откройте Chrome/Chromium → `chrome://extensions/`
4. Включите "Режим разработчика"
5. Нажмите "Загрузить распакованное расширение" и выберите папку с расширением

## Быстрый старт

1. Нажмите на иконку расширения для открытия popup
2. Используйте переключатель для включения/выключения расширения
3. Выделите текст и нажмите правой кнопкой мыши → Расширение → Перевести или Объяснить
4. Настройте параметры ИИ на странице настроек

## Требования

- Браузер на базе Chromium (Chrome, Edge, Brave, Opera и т.д.)
- API ключ для модели ИИ (DeepSeek или OpenAI)
- Интернет-соединение
