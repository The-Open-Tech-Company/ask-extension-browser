# Инструкция по установке

## Быстрый старт

### Шаг 1: Создание иконок

Перед установкой расширения необходимо создать иконки. У вас есть два варианта:

#### Вариант 1: Использование HTML генератора (рекомендуется)
1. Откройте файл `create-icons.html` в браузере
2. Нажмите кнопку "Скачать все иконки"
3. Сохраните файлы `icon16.png`, `icon48.png`, `icon128.png` в папку `icons/`

#### Вариант 2: Использование Python скрипта
1. Убедитесь, что установлен Python 3
2. Установите библиотеку Pillow: `pip install Pillow`
3. Запустите скрипт: `python generate_icons.py`
4. Иконки будут созданы в папке `icons/`

#### Вариант 3: Создание вручную
Создайте три PNG файла:
- `icons/icon16.png` (16x16 пикселей)
- `icons/icon48.png` (48x48 пикселей)
- `icons/icon128.png` (128x128 пикселей)

### Шаг 2: Установка расширения

1. Откройте браузер Chrome/Chromium/Edge
2. Перейдите по адресу:
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
   - Другие Chromium браузеры: `chrome://extensions/` или найдите в настройках
3. Включите "Режим разработчика" (Developer mode) в правом верхнем углу
4. Нажмите "Загрузить распакованное расширение" (Load unpacked)
5. Выберите папку `browaskelex` (папку с файлами расширения)
6. Расширение будет установлено и готово к использованию

### Шаг 3: Настройка

1. Нажмите на иконку расширения в панели инструментов браузера
2. Перейдите на вкладку "Настройки AI"
3. Выберите модель ИИ (DeepSeek или ChatGPT)
4. Введите API ключ:
   - **DeepSeek**: https://platform.deepseek.com/api_keys
   - **ChatGPT**: https://platform.openai.com/api-keys
5. Настройте температуру и системный промпт при необходимости
6. Нажмите "Сохранить"

### Шаг 4: Использование

#### Перевод текста:
1. Выделите текст на любой веб-странице
2. Правой кнопкой мыши → "Расширение" → "Перевести"
3. Рядом с текстом появится окно с переводом

#### Объяснение текста:
1. Выделите текст на любой веб-странице
2. Правой кнопкой мыши → "Расширение" → "Объяснить"
3. Откроется окно чата с ИИ
4. ИИ автоматически объяснит текст
5. Вы можете задавать дополнительные вопросы

## Требования

- Браузер на базе Chromium (Chrome, Edge, Brave, Opera и т.д.)
- API ключ для выбранной модели ИИ
- Интернет-соединение

## Устранение проблем

### Расширение не загружается
- Убедитесь, что все файлы на месте
- Проверьте, что иконки созданы в папке `icons/`
- Проверьте консоль браузера на наличие ошибок (F12)

### Переводчик не работает
- Проверьте интернет-соединение
- Убедитесь, что сайт не блокирует запросы
- Попробуйте выделить текст заново

### ИИ не отвечает
- Проверьте правильность API ключа
- Убедитесь, что ключ активен и имеет баланс (для платных API)
- Проверьте сообщение об ошибке в окне чата
